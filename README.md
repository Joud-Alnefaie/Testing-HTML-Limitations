## Originally an Internet Technologies course project requiring 3 pages. Expanded to 37 pages to explore the limits of HTML to see whether the gameplay and flow of visual novel games can be implemented solely using HTML. At first it seemed like it can be achieved by using HTML only yet, as I progressed, more and more limitations arose so, there were some places where I was forced to use CSS.
